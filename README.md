## 🚀 How to run

해당 프로젝트를 처음 빌드하시는 경우 아래 명령어를 실행해주세요.
필요한 파이썬 패키지들을 설치합니다.
```
./scripts/installation.sh
```

설치가 된 상태라면 아래 명령어를 통해 최신 버전의 문서를 확인하실 수 있습니다.
8080번 포트에서 웹 서버가 띄워집니다.

```
./scripts/run.sh
```
